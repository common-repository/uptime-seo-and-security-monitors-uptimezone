== Installation ==

* Install and activate the UptimeZone Plugin through the 'Plugins' menu in WordPress
* After installation, click the Settings → UptimeZone section in the left toolbar
* Create a new free UptimeZone account, or use your existing one. Log in into https://uptimezone.com in another browser tab and copy the widget code from "Your profile" section.
* Go back to Wordpress console, paste the code and press Save changes
* Go back to UptimeZone dashboard and create your new monitors
* Done!
